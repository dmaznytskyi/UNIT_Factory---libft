## On UNIX systems (compilation & usage)

Compile like this:

```sh
$ make
```

Use with your project:

- Include header: '#include "/path/to/folder/libft.h"'
- Compile
```sh
$ gcc <your flags> -I /path/to/folder/libft/ -L libft/ -lft
```
